# user-app-axios

Extended version of user-app that fetches user data from an API using Axios instead of local state.

## Context
- **Course:** Ksquare University Bootcamp
- **Date:** 2021

## Tech Stack
- React 17 (hooks, Context API)
- Axios
- Material UI
- Create React App

## What It Does
Same functionality as user-app (login, signup, user cards, theme toggle) but uses Axios to fetch user data from an external API. Demonstrates the progression from hardcoded data to API-driven data fetching while maintaining the same UI with Material UI components and Context-based state.
